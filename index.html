Vanila Javascript

Vanilla JavaScript refers to using pure JavaScript without any libraries or frameworks. It gives full control over the DOM, browser APIs, and core logic to build dynamic web functionality.

React

Node.js

Node.js is a JavaScript runtime that lets you run JS on the server side. It’s great for building fast, scalable backend services using an event-driven, non-blocking architecture.

MongoDB

MongoDB is a NoSQL database that stores data in flexible JSON-like documents. It allows easy scaling and fast data operations, making it ideal for modern web applications
